API Documentatie

Nieuws Endpoints

In url api/news of users

GET /news

Haalt alle nieuwsartikelen op.

GET /news/:id

Haalt een nieuwsartikel op basis van ID.

Parameters:

  • id - ID van het nieuwsartikel.

POST /news

Voegt een nieuw nieuwsartikel toe.

Request Body:

    {
        "title": "Breaking News",
        "content": "This is a breaking news article",
        "published_at": "2025-01-13"
    }

PUT /news/:id

Werkt een bestaand nieuwsartikel bij.

Parameters:

  • id - ID van het nieuwsartikel.

Request Body:

    {
        "title": "Updated News Title",
        "content": "This is the updated content of the news article.",
        "published_at": "2025-01-14"
    }

DELETE /news/:id

Verwijdert een nieuwsartikel.

Parameters:

  • id - ID van het nieuwsartikel.

Users Endpoints

GET /users

Haalt alle gebruikers op.

GET /users/:id

Haalt een gebruiker op basis van ID.

Parameters:

  • id - ID van de gebruiker.

POST /users

Voegt een nieuwe gebruiker toe.

Request Body:

    {
        "name": "John Doe",
        "email": "johndoe@example.com",
        "password": "securepassword",
        "birthday": "1990-01-01",
        "about": "I am a new user",
        "role": "user",
        "created_at": "2025-01-13"
    }

PUT /users/:id

Werkt een bestaande gebruiker bij.

Parameters:

  • id - ID van de gebruiker.

Request Body:

    {
        "name": "John Doe",
        "email": "johndoe@example.com",
        "password": "newsecurepassword",
        "birthday": "1990-01-01",
        "about": "Updated information",
        "role": "user",
        "created_at": "2025-01-14"
    }

DELETE /users/:id

Verwijdert een gebruiker.

Parameters:

  • id - ID van de gebruiker.
